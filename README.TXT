PROJECT TITLE: MINGO RUSH
PURPOSE OF PROJECT: 
- to practice and develop programming skill

HOW TO START THIS PROJECT: define type of game and draw storyboard
AUTHORS: Naruhpak Rotchanakanokchok
USER INSTRUCTIONS: 
Player is flamingo named Mingo
How to control
- press up  to jump
- press left to move backward
- press right to move forward
How to play 
- control Mingo to avoid dragon fire, crystal, and T-Rex
- collect beetroot to get special score (1 beetroot = 50 score)
- score will increasing by distance that Mingo can survive
